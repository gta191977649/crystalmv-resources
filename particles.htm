Particle objects documentation

Go back to Particle objects page

Particle objects resource works by adding custom models which only contain the particle data. Therefore particle effects are added by creating objects with particle model IDs. Some effects have a limited time after which they stop appearing. If you want to make them visible continuously, you need to keep destroying and recreating the object.

Example

The following example creates the riot smoke in the center of the map. Because the effect time is limited, the object has to be recreated repeatedly.

function createRiotSmoke()
    smoke = createObject(2054, 0, 0, 3)
end

function recreateRiotSmoke()
    destroyElement(smoke)
    createRiotSmoke()
end

createRiotSmoke()
setTimer(recreateRiotSmoke,5000,0)
